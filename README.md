# Posteena

`Posteena` is a web application multitool designed for post-sharing, personal note-taking, and password management. It is built with a strong focus on anonymity and user-controlled storage, supporting various adapters to save data to different cloud drive services.

## Project Guidelines & Contribution Rules

To maintain high code quality and consistency, all contributors (including LLMs) must follow the project's core rules.

**The full set of rules is documented in [RULES](RULES.md).**
